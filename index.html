What is Campion.js?

Campion.js is a constraint satisfaction program written entirely in JavaScript!

Campion uses constraint propogation to solve a wide variety of problems from many business and personal domains

Download from GitHub!

Constraint Propagation?

Constraint Propagation Problems are nothing more a set of variables, a set of possible values for each variable, and relationships between the variables that constrain the variables.

  • Variables - These are your nouns. For example: CathyAge or boardspace0_1, or NumberOfMarblesICanFitInMyMouth
  • Domains - These are the values your values can take. For Example : 50-59, 0-9, or 15-20, each domain belongs to a particular variable
  • Constraints - These are relationships between your variables, For Example: CathyAge < TomAge if Cathy is younger than Tom

The constraint solver will get to work. A value will be picked from the domain of the first variable. Then each constraint will propagate to the other variables, and what that choice for the first variable means for the rest of them. If Tom's age is narrowed down to 55 and campion knows Cathy's Age is less than than toms then campion can narrow Cathy's Domain down from 50-59 to 50-54.
